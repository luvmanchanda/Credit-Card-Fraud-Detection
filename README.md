Credit Card Fraud Detection | Classification Machine Learning Project

Developed an end-to-end Classification Machine Learning project to detect fraudulent credit card transactions by applying the complete machine learning workflow, from data collection to model evaluation.

The project began with collecting a real-world credit card transaction dataset from Kaggle, followed by data cleaning, preprocessing, and preparing the dataset for model training. The objective was to accurately classify transactions as either fraudulent or legitimate using supervised machine learning techniques.

The model development phase involved implementing Logistic Regression, a widely used algorithm for binary classification problems. The model was trained, tested, and evaluated on unseen data, achieving an Accuracy Score of 96.88% on the test dataset, demonstrating strong performance in identifying fraudulent transactions.

The entire project was developed using Python in Google Colab, leveraging libraries including NumPy, Pandas, Matplotlib, and Scikit-learn. The complete source code has been published on GitHub, showcasing the complete development process, model training, and evaluation.
